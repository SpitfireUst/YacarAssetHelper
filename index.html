Help

To update an Asset:

  • Search by Id in the Asset Editor
  • After updating the info click the `Update Asset` button
  • Then click the copy asset.json button
  • Navigate to https://github.com/coinhall/yacar
  • Create a new PR and replace the asset.json file from github with your copied file

To create a new Entity (add social links to Asset):

  • Fill in the data in the Create New Entity Editor
  • Put in the contract address(es) related to the newly created entity in the Asset IDs field
  • Press the `Update Entity and Assets` button
  • Then click the copy asset.json button
  • Navigate to https://github.com/coinhall/yacar
  • Create a new PR and replace the asset.json file from github with your copied file
  • Press the copy entity.json button
  • Replace the entity.json file from github with your copied file
